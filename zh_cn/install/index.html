安装

安装

下载基础文件:

  • CrossC2.cna
  • genCrossC2 CS运行环境
  1. 拷贝 CrossC2.cna 与 genCrossC2 文件到CobaltStrike 根目录 下 (必须处于 同目录 )
  2. 选择Script Manager，添加CrossC2.cna (如果成功安装，菜单栏会多出一项 CrossC2)
  3. 修改CrossC2.cna脚本中genCrossC2路径为真实路径
exec("/xxx/xxx/genCrossC2"... -> exec("/opt/cs/genCrossC2"...
