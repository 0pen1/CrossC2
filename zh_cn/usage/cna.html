cna插件

.cna 使用方法

菜单栏: CrossC2 -> CrossC2 Payload Generator -> genCrossC2

弹出的对话框中可以配置:
1. 操作系统
2. 运行位数
3. Payload类型(目前仅支持Stageless, Staged正在更新中)
4. 生成文件保存路径

最终将生成Payload到指定目录以及创建供wget等工具使用的Scripted Web Delivery
